# Ejemplos aprobados

## Estado

La auditoría no confirmó todavía un corpus formal de artículos o carruseles aprobados que pueda publicarse como ejemplo normativo.

No se deben presentar documentos históricos, carruseles existentes o ejemplos del renderer como aprobados sin confirmación del administrador.

## Plantilla para incorporar ejemplos

Cada ejemplo futuro debe indicar:

- identificador del artículo o carrusel;
- versión;
- tipo de texto o slide;
- qué criterio editorial satisface;
- qué criterio visual satisface;
- si fue aprobado para reutilización como ejemplo del GPT.
