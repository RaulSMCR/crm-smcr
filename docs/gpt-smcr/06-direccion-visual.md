# Dirección visual

## Fuentes

- Requisitos visuales explícitos del administrador.
- `IDENTIDAD-VISUAL.md`.
- `tailwind.config.js`.
- `src/app/globals.css`.
- `api/slides/_lib/renderer.py`.
- `api/slides/_assets/logo.png`.
- Partes visuales de `vendor/instagram-slides/SKILL.md`.

## Reglas confirmadas

- Estilo editorial, educativo, sereno y no vendedor.
- Logo de hoja de monstera en la esquina inferior izquierda.
- Texto integrado directamente sobre el arte.
- Evitar paneles de color que oculten la composición.
- Usar la paleta actual del proyecto.
- Mantener tratamiento consistente de título, cuerpo y cierre.
- Mantener legibilidad y coherencia entre slides.

## Paleta de trabajo

- `#2B7073` teal.
- `#163A3C` teal oscuro.
- `#FB7A62` coral.
- `#B85442` coral oscuro.
- `#F6EFDF` crema.

La paleta procede de `IDENTIDAD-VISUAL.md` y de la configuración visual auditada. Debe permanecer sincronizada con el renderer antes de convertirse en una validación automática.

## Separación de responsabilidades

- El manifiesto editorial decide el contenido y sus límites.
- Esta dirección decide su presentación visual.
- El renderer decide el resultado técnico ejecutable.
- Una decisión visual no puede alterar el grado de evidencia de un texto.
