# Identidad y marca

## Fuente y autoridad

- Requisitos explícitos del administrador: dominio, logo, ubicación y carácter visual.
- `IDENTIDAD-VISUAL.md`: documento visual de trabajo sin seguimiento en Git.
- `tailwind.config.js` y `src/app/globals.css`: implementación visual existente.
- `api/slides/_lib/renderer.py` y `api/slides/_assets/logo.png`: implementación visual del renderer.

Cuando exista una diferencia, los requisitos explícitos del administrador gobiernan este documento.

## Identidad

- Dominio exacto: `SaludMentalCostaRica.com`.
- Logo: hoja de monstera.
- Ubicación fija del logo: esquina inferior izquierda.
- Carácter: editorial, educativo, sereno y no vendedor.
- El texto forma parte directamente del arte.
- No usar paneles de color que oculten la composición.
- Mantener una presentación consistente entre títulos, cuerpo y cierre.

## Regla de separación

La identidad visual decide cómo presentar el contenido. El manifiesto editorial decide qué contenido puede afirmarse y cómo debe matizarse.
