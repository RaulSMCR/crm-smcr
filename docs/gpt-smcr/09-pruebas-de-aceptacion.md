# Pruebas de aceptación

## Editoriales

- El texto diferencia lectura y examen.
- El grado de evidencia no se exagera.
- No aparece `algunos estudios sugieren`.
- La falta de investigación no se convierte en veredicto.
- La experiencia clínica no se usa como prueba de eficacia.
- No se explican prácticas por intenciones o psicología de adherentes.
- La distancia no se traza mediante sarcasmo o descalificación moral.
- No hay diagnóstico prestado ni CTA de urgencia.
- El carrusel deja criterios o preguntas útiles para el lector.

## Copy de slides

- Cada slide tiene una idea principal.
- La reducción de extensión no elimina una condición que cambie el sentido.
- La secuencia conserva el argumento central.
- Los textos resultan comprensibles para un lector culto no especialista.
- Las referencias se conservan en el artículo o metadata cuando corresponda.

## Visuales

- Se usa la paleta documentada.
- El logo es la hoja de monstera.
- El logo permanece en la esquina inferior izquierda.
- El texto está integrado en el arte.
- No se usan paneles que oculten la composición.
- Títulos, cuerpo y cierre conservan tratamiento consistente.
- La presentación es editorial, educativa, serena y no vendedora.

## Intercambio y CRM

- La exportación contiene los archivos definidos.
- El JSON inválido se rechaza.
- Los errores indican slide o campo cuando corresponde.
- La importación muestra diferencias antes de guardar.
- La vista previa no modifica la base de datos.
- La confirmación crea una versión nueva.
- Una versión anterior permanece disponible.
- Editar texto no reemplaza el asset.
- Reemplazar una imagen no modifica las demás slides.
- Aprobar o rechazar una slide no cambia las otras.
- Una regeneración selectiva no elimina assets aprobados ajenos.
